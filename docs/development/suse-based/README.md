# Development on SUSE-Based Systems

This section contains guides for setting up development tools on SUSE-based distributions like openSUSE Leap and openSUSE Tumbleweed.

These distributions are known for their robustness, the powerful YaST administration tool, and the Open Build Service (OBS) for package availability.
