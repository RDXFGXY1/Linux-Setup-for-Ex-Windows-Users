# Command-Line on Arch-Based Systems

This section provides guides for common command-line tasks on Arch-based distributions like Arch Linux, Garuda, and EndeavourOS.

The primary package manager is `pacman`, and the Arch User Repository (AUR) is typically accessed with an AUR helper like `yay`.
